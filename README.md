# Soil Benchmarking

This repository hosts scripts and data for **soil benchmarking analyses** using the **LUCAS dataset**. The goal is to evaluate soil properties and benchmark soil quality indicators across European sites, supporting reproducible workflows for soil data exploration and visualization.

---

## Repository structure

- **Figures/**  
  Contains plots and visual outputs generated from benchmarking analyses.

- **test_plots.R**  
  Example script for testing soil benchmarking curves using the LUCAS dataset.

---

## Getting started

1. Clone the repository:
   ```bash
   git clone https://github.com/Luciana-cloud/soil_benchmarking

2. Open the project in RStudio using `LUCAS_data.Rproj`.

3. Run `test_plots.R` to generate preliminary benchmarking plots.

4. Modify or extend scripts to include additional soil indicators or datasets.
